## 介绍📖
Meteor System 是一个极简的后台管理系统解决方案，基于 Vue3 和 Ant Design Vue 实现。使用了最新的前端技术栈 Vue3 Setup语法糖 & Vite3 & Vue-Router4 & Ant Design Vue & Pinia & TypeScript 。

后台SpringBoot项目 Meteor，链接：https://github.com/huangjiayao1993/Meteor
